Splatoon: Filetypes

These are all the filetypes in the content folder, and links to tools on how to edit them.

Types:

Packages:

  • .szs

This is a compressed file format, and can contain other files inside of it.

  • Switch Toolbox

  • .pack

This is a file format that contains other files and folders inside of it, without direct compression on the files.

  • Switch Toolbox

Sound:

  • .bfstm

This is a filetype used for soundtracks, like battle music, and is streamed to RAM.

  • Switch Toolbox
  • BrawlBox

  • .bfwav

This is used for sound effects in Splatoon, usually found in .bars files.

  • Read .bfwav: Switch Toolbox
  • Create .bfwav: (placeholder)

  • .bameta

These files seem like they are metadata files for their corresponding .bfwav files. (Through minimal testing, I could not figure out how much control the file has on how the sound effect is played.)

  • Hexed.it (no good editor yet)

  • .bars

This is used to contain and package .bfwav files with their .bameta files, and section them off to related objects/scenes.

  • Switch Toolbox

  • .btsnd

This is Splatoon’s bootsound, found in the meta folder. This covers the transition between the Wii U menu .rpx, and the Splatoon .rpx.

  • wav2btsnd

Models:

  • .bfres

This is Splatoon’s model file, which can contain models, materials, textures, animations, and more.

  • Switch Toolbox

Other:

  • .bin (only for PlazaNpcPreset??.bin)

This is a binary file, holding data for the original 30 Plaza NPCs.

  • PlazaNpcPreset-Editor v1.0
