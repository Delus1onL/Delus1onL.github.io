Splatoon: “/content/Model”

This is all the sections of model types used in the game. These are all

.szs compressed files.

Files:

  • Clothing

These are the files for the shirts, or “clothes.” The name for all these start with “Clt_”.

  • Enemys

These are the files for the enemy bosses, found in the story mode, or “Octo Valley.” The name for all these start with “Enm_”.

  • Fields

These are the files for the fields, or “maps.” The name for all these start with “Fld_”.

  • Headgear

These are the files for the headgear, or “hats.” The name for all these start with “Hed_”.

  • Lifts

These are the files for the lifts, or moveable objects in the game. The name for all these start with “Lft_”.

  • NPCs

These are the files for the NPCs, like Callie, Marie, shopkeepers, and more. The name for all these start with “Npc_”.

  • Objects

These are the files for the objects not found in Obj.pack. The name for all these start with “Obj_”.

  • Shoes

These are the files for the shoes. The name for all these start with “Shs_”.

  • Skyboxes

These are the files for most of the skyboxes. The name for all these start with “VR_”.

  • Weapons

These are the files for the weapons added in updates, after the base release. The name for all these start with “Wmn_”, except for the rainmaker, which is “Wsp_Shachihoko”.

  • Other

These are the other model files. The don’t have enough in their category to be given their own page, but won’t be left unmentioned. This includes “Cvy_” and “DrcMap_”.
